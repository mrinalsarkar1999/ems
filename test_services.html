🔧 Service Status Checker

This page helps you verify that all required services are running for the attendance tracking system.

📋 Required Services

Make sure these services are running:

  • React Frontend: npm start (Port 3000)
  • Node.js Backend: npm run server (Port 5000)
  • Face Auth Server: python -m uvicorn app:app --host 0.0.0.0 --port 8000 --reload (Port 8000)

🔍 Manual Checks

If services are not running, check:

  • Are all terminal windows open and running the correct commands?
  • Are there any error messages in the terminal?
  • Are the correct ports (3000, 5000, 8000) available?
  • Is MongoDB running and accessible?
  • Are AWS credentials configured in .env file?
